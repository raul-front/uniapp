# uniapp-base
课观uniapp基础框架

## 微信平台
> 运行开发环境
```
npm run dev
```
> 打包测试环境
```
npm run test
```
> 打包预发环境
```
npm run pre
```
> 打包生产环境
```
npm run release
```

## 微信开发者工具导入项目
> 开发项目
```
dist/weixin/dev
```
> 测试项目
```
dist/weixin/test
```
> 预发项目
```
dist/weixin/pre
```
> 生产项目
```
dist/weixin/release
```

## 上传微信代码包
> 测试环境
```
make wx-test
```
> 预发环境
```
make wx-pre
```
> 生产环境
```
make wx-release
```
